## 介绍

一个简单的命令行翻译工具，仅此而已

## 动机

- 绝大部分AI工具没有开箱即用的翻译工具
- 终端工作流的情况下需要反复切换页面

## 安装

暂时仅支持本地构建

## 用法

```Shell
transome [text]
```

程序将读取环境变量中的**GOOGLE_AI_API_KEY**作为Key, 使用**gemini-2.5-flash-lite-preview-06-17**模型，当然你也可以手动设置

执行以下代码获取更多信息

```
transome --help
```
